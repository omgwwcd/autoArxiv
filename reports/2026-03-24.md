### 2. 为什么重要

企业AI代理在真实部署中受限于数据质量差、数量少、复杂推理需求和缺乏可靠反馈信号，该框架提供了一种轻量级、模型无关的离线强化学习方法，可显著提升代理性能并具有良好的泛化能力。

### 4. 背景与问题设定

## 3. GSEM: Graph-based Self-Evolving Memory for Experience Augmented Clinical Reasoning

### 1. 一句话结论

GSEM提出了一种基于图的自演化记忆框架，通过双层图结构组织临床经验，支持可应用性感知检索和在线反馈驱动的节点质量与边权重校准，显著提升了临床决策代理的准确性和可靠性。

### 2. 为什么重要

现有记忆增强方法通常将经验存储为独立记录，缺乏显式关系结构，导致检索噪声、重用不可靠，甚至可能损害性能。GSEM通过构建双层记忆图，显式捕捉经验内部决策结构和跨经验关系依赖，支持可应用性感知检索和在线反馈驱动的校准，有效解决了边界感知和关系感知的挑战，显著提升了临床决策代理的性能。

### 3. 研究问题

1. 如何有效组织临床经验以支持可应用性感知的检索？
2. 如何建模跨经验的关系依赖以支持协同决策？
3. 如何在不修改经验内容的前提下，通过在线反馈持续优化记忆的可靠性和结构？

### 4. 背景与问题设定

临床决策代理可以从重用先前的决策经验中受益。然而，许多记忆增强方法将经验存储为独立的记录，缺乏显式的关联结构，这可能导致噪声检索、不可靠的重用，甚至在某些情况下会损害性能。现有方法在处理边界感知（如忽略关键条件）和协作失败（如共检索到冲突的经验）方面存在挑战。

### 5. 方法概览

- 我的理解: GSEM的核心创新在于将经验组织成一个双层图结构，其中实体层捕捉每个经验的内部决策流，经验层捕捉经验之间的关系。通过混合种子召回和LLM引导的多种子图遍历，实现边界感知和关系感知的检索。更重要的是，它通过在线反馈驱动的校准机制，持续优化节点质量和边权重，而无需修改经验内容，从而实现了记忆的自演化。

## 4. Agentic Personas for Adaptive Scientific Explanations with Knowledge Graphs

**Topics:** Agent Skill, Memory, RL, Evolve
**Authors:** Susana Nunes, Tiago Guerreiro, Catia Pesquita
**Venue / Year:** 
**Links:** [Abstract](http://arxiv.org/abs/2603.21846v1) | [PDF](http://arxiv.org/pdf/2603.21846v1.pdf)

### 1. 一句话结论

通过引入代理人格（agentic personas）和强化学习，该研究实现了基于知识图谱的科学解释的自适应生成，能够根据专家的不同认知策略生成个性化解释，显著减少对人类反馈的依赖并提升解释的可接受性。

### 2. 为什么重要

在药物发现等高风险科学领域，现有解释方法通常假设静态用户模型，无法适应专家多样化的认知策略和知识立场，导致解释缺乏深度和针对性。该研究通过代理人格和强化学习框架，实现了可扩展的自适应可解释性，显著降低了对人类反馈的需求，并提升了解释的科学有效性和用户接受度。

### 3. 研究问题

1. 如何在知识图谱解释中实现自适应，以适应专家多样化的认知策略和知识立场？
2. 如何在缺乏大量个体用户数据的情况下，构建可代表专家解释偏好的结构化代理人格？
3. 代理人格驱动的解释生成能否在保持预测性能的同时，更符合专家的偏好？

### 4. 背景与问题设定

当前基于知识图谱的解释方法虽然能进行基于路径的推理，但通常假设一个静态的通用用户模型，无法适应专家在科学发现中不同的认知策略和知识立场（epistemic stances）。这导致生成的解释无法促进更深层次的理解和知情决策。然而，专家数量有限，直接使用人类反馈来训练自适应解释系统成本高昂。

### 5. 方法概览

- 任务 / 环境: 药物发现领域，具体为药物重定位和药物-靶点相互作用预测任务，使用Hetionet异构生物医学知识图谱。
- 变量 / 干预设计: 提出了一种两阶段方法：(I) 从专家反馈中创建代理人格（agentic personas），通过嵌入、聚类和大型语言模型（LLM）合成来构建代表不同解释偏好的结构化代理人格；(II) 在强化学习（RL）框架中，使用代理人格作为奖励模型，引导解释生成，使其适应特定的解释偏好。
- 评价指标: ['预测性能（链接预测准确率）', '用户偏好（在22名参与者中的偏好选择）', '解释质量（有效性、相关性、完整性评分）', '与专家评估的一致性（Persona评分与专家评分的相关性）']
- 模型比较: 将提出的代理人格驱动的自适应解释模型与非自适应基线（REx）进行比较，评估其在预测性能和用户偏好上的表现。
- 我的理解: 该研究巧妙地将人类专家的解释偏好抽象为“代理人格”，并将其作为强化学习的奖励函数，从而在不直接追踪个体用户的情况下，实现了对不同解释风格的自适应。这解决了数据稀缺问题，并通过LLM合成和聚类，将专家反馈转化为可操作的、可解释的代理人格，为构建更智能、更人性化的AI解释系统提供了新范式。

### 6. 关键发现

- 代理人格有效捕捉专家多样性: 通过聚类分析，研究成功识别出两个代表不同知识立场的代理人格：Elena（强调机制严谨性）和Leo（偏好高阶概念）。代理人格的评分与原始专家评分高度一致，证明其能有效作为专家解释偏好的代理。
- 自适应解释更受用户欢迎: 在22名参与者的用户研究中，代理人格驱动的自适应解释在有效性、相关性和完整性上均优于非自适应基线，并且在偏好选择上显著更受青睐。
- 显著降低反馈需求: 与传统方法相比，基于代理人格的训练将反馈需求减少了两个数量级，实现了可扩展的自适应可解释性。
- 保持预测性能: 代理人格驱动的解释模型在药物重定位任务上的链接预测性能与最先进的基线模型相当，证明了其在生成高质量解释的同时，没有牺牲预测能力。

- 对 Agent 系统: 为AI代理系统提供了构建自适应、个性化解释能力的新范式，使其能够根据不同的用户（如专家、临床医生）生成更符合其认知风格的解释。

- LLM在生成人格叙事时可能存在偏差或不真实性。

## 5. TREX: Trajectory Explanations for Multi-Objective Reinforcement Learning

**Topics:** Agent, Skill, RL, Explainable AI, Multi-Objective Reinforcement Learning
**Authors:** Dilina Rajapakse, Juan C. Rosero, Ivana Dusparic
**Venue / Year:** 
**Links:** [Abstract](http://arxiv.org/abs/2603.21988v1) | [PDF](http://arxiv.org/pdf/2603.21988v1.pdf) | [Code](https://github.com/dilina-r/trex_xmorl)

### 1. 一句话结论

TREX通过轨迹归因量化多目标强化学习策略中不同行为模式对目标权衡的影响，揭示策略决策背后的可解释性逻辑。

### 2. 为什么重要

多目标强化学习（MORL）在现实世界中广泛应用，但其决策过程常被视为黑箱。现有可解释性方法主要针对单目标奖励，无法解释多目标权衡。TREX填补了这一空白，提供了一种量化分析多目标策略中行为模式影响的框架，增强了模型的透明度和可信度。

### 3. 研究问题

1. 如何解释多目标强化学习策略中的决策过程？
2. 如何量化不同行为模式对多目标权衡的影响？
3. 如何识别和解释策略中影响目标权衡的关键行为模式？

### 4. 背景与问题设定

强化学习（RL）在解决复杂决策问题上表现出色，但许多现实场景涉及多个相互冲突的目标，无法用单一标量奖励表示。多目标强化学习（MORL）通过同时优化多个目标来解决此问题，但其“黑箱”特性使得所选目标权衡的决策过程不清晰。现有的可解释强化学习（XRL）方法通常针对单标量奖励，不考虑与不同目标或用户偏好相关的解释。

### 5. 方法概览

- 任务 / 环境: 多目标MuJoCo环境，包括MO-HalfCheetah、MO-Ant和MO-Swimmer。
- 变量 / 干预设计: TREX框架通过生成专家策略在不同用户偏好下的轨迹，将轨迹聚类为语义上有意义的时间段。然后，通过训练排除特定聚类的互补策略，测量相对于原始专家策略的观测奖励和动作的相对偏差，来量化这些行为段对帕累托权衡的影响。
- 评价指标: 相对奖励变化（ΔRi）、总奖励偏差（ΔR）、奖励归因分数（RAS）
- 模型比较: 
- 我的理解: TREX的核心思想是将策略的决策过程分解为可解释的行为模式。通过分析排除特定行为模式后策略性能的变化，可以量化这些模式对目标权衡的贡献。这种方法超越了定性观察，提供了对策略行为的定量分析。

### 6. 关键发现

- TREX能够量化行为模式对目标权衡的影响: 通过训练排除特定聚类的互补策略，TREX能够量化这些行为模式对每个目标和整体目标权衡的影响。例如，在MO-HalfCheetah中，'长跃'行为最大化速度但增加能耗，而'短步'则相反。TREX可以量化这些行为对不同用户偏好的影响。
- TREX能够识别关键行为模式: TREX通过奖励归因分数（RAS）识别出对目标权衡影响最大的行为模式。高RAS分数表明该行为模式对目标权衡的转变有显著影响。

### 10. 对 Agent / Skill / Memory 的启发

- 对 Agent 系统: TREX框架可以用于增强多目标强化学习代理的透明度和可信度，使其决策过程更易于理解和验证。
- 对 Skill: TREX可以用于分析和理解代理在多目标任务中学习到的技能，识别出哪些技能对特定目标的实现至关重要。
- 对 Memory: TREX通过分析轨迹，可以揭示代理在学习过程中记忆和利用特定行为模式的方式。
- 对评测: TREX提供了一种新的评估多目标强化学习策略的方法，不仅评估最终性能，还评估策略决策过程的可解释性。

### 11. 局限性

- TREX依赖于专家策略生成的轨迹，其解释能力受限于专家策略的质量。
- 聚类算法的选择和参数设置可能影响结果的准确性。
- 计算成本较高，需要训练多个互补策略。

### 12. 我的看法 / 个人笔记

TREX是一个创新且实用的框架，它通过量化行为模式的影响，为多目标强化学习的可解释性提供了一个新的视角。它不仅能够解释策略的决策过程，还能帮助理解不同行为模式在实现多目标权衡中的作用。

### 14. 总结

TREX是一种基于轨迹的可解释性框架，用于解释多目标强化学习策略。它通过生成专家策略在不同用户偏好下的轨迹，将轨迹聚类为语义上有意义的时间段，并通过训练排除特定聚类的互补策略来量化这些行为段对帕累托权衡的影响。实验结果表明，TREX能够有效隔离和量化特定的行为模式，为理解多目标强化学习策略的决策过程提供了新的见解。
