## 2. Box Maze: A Process-Control Architecture for Reliable LLM Reasoning

- arXiv ID: `2603.19182v1`
- Topics: NLP, Agent Skill Memory
- Why read: 该论文提出的Box Maze框架通过显式的过程控制层，为解决NLP中的幻觉问题和提升Agent的技能记忆可靠性提供了创新的架构思路。
- Abstract: http://arxiv.org/abs/2603.19182v1
- PDF: http://arxiv.org/pdf/2603.19182v1.pdf

该论文针对大语言模型（LLMs）在对抗性提示下易产生幻觉和不可靠推理的问题，提出了一种名为Box Maze的过程控制架构。该架构将推理过程分解为三个显式层：记忆锚定（Memory grounding）、结构化推理（Structured inference）和边界强制（Boundary enforcement），旨在从架构层面确保推理过程的完整性。通过在多个异构LLM系统（DeepSeek-V3, Doubao, Qwen）上进行模拟评估，结果表明，与基线RLHF方法（边界失败率约40%）相比，Box Maze框架在对抗性场景下将边界失败率降低到1%以下，显著提升了推理的一致性和可靠性。该工作为通过过程级控制增强LLM推理可靠性提供了新的方向。

## 3. What Really Controls Temporal Reasoning in Large Language Models: Tokenisation or Representation of Time?

- Abstract: http://arxiv.org/abs/2603.19017v1
- PDF: http://arxiv.org/pdf/2603.19017v1.pdf

该研究提出了MultiTempBench，一个涵盖五种语言（英语、德语、中文、阿拉伯语和豪萨语）和三种历法（格里高利历、希吉拉历和中国农历）的多语言时间推理基准，包含15,000个示例，用于评估日期算术、时区转换和时间关系提取任务。研究评估了20个大语言模型，并引入了多语言日期碎片化比率（mDFR）和几何探测分析，以量化时间表示的质量。主要发现是：在低资源语言和罕见历法中，时间实体的分词质量是关键瓶颈，碎片化会破坏年/月/日的分离，导致准确率下降；而在高资源语言中，时间线性是时间推理的最强预测因子。这表明时间推理的控制机制在不同语言资源水平下存在差异，分词控制表面访问，而时间线性控制内部表示。

## 5. Hypothesis-Conditioned Query Rewriting for Decision-Useful Retrieval

- arXiv ID: `2603.19008v1`
- Topics: NLP, Agent Skill Memory
- Why read: 该论文提出的HCQR框架通过假设条件查询重写，有效提升了RAG在决策任务中的检索质量，对NLP和Agent Skill Memory领域具有重要参考价值。
- Abstract: http://arxiv.org/abs/2603.19008v1
- PDF: http://arxiv.org/pdf/2603.19008v1.pdf

该论文针对检索增强生成（RAG）在多选项决策任务中因检索到的上下文偏向主题相关性而非决策相关性而导致的性能不足问题，提出了一种无需训练的预检索框架Hypothesis-Conditioned Query Rewriting (HCQR)。HCQR首先从问题和候选选项中生成一个轻量级工作假设，然后将其重写为三个目标明确的查询：支持该假设的证据、区分其与竞争选项的证据，以及验证问题关键线索的证据。这种方法使检索更直接地对齐于答案选择，从而帮助生成器基于检索到的证据确认或推翻初始假设。在MedQA和MMLU-Med数据集上的实验表明，HCQR在平均准确率上分别比Simple RAG提升了5.9和3.6个百分点，显著优于单查询RAG和重排序/过滤基线方法。
